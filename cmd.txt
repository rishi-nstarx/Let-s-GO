# Memory management

--> Memoy allocation and deallocation happens automatically.
    GC (Garbage collection) happens automatically.
    Out of scope or nil.

new(): allocate memory but not INIT (initialization).
        you will get a memory address.
        zeroed storage (can't put any data initially).

make(): allocate memory but INIT (initialization).
        you will get a memory address.
        non-zeroed storage (can put any data initially).
